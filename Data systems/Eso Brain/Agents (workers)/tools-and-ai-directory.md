## TOOL DECISION GUIDE

```
Need to think deeply or plan?          → Claude
Need to write a script or code?        → DeepSeek
Need to write content or templates?    → ChatGPT
Need current research or facts?        → Grok or Perplexity
Need to control your PC?               → OpenClaw or PowerShell
Need automation workflows?             → n8n
Need to edit many files at once?       → VS Code
Need to search your own notes with AI? → Ollama + Smart Connections
Need a YouTube video summarized?       → Turbo AI
Need voice input?                      → Whisper AI
Need voice output?                     → ElevenLabs
Need Google Calendar connected?        → Gemini
Need tasks and reminders?              → TickTick
```
