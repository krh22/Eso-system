# ESO Data Inventory
**Status**: v0.1 (Comprehensive Mapping)
**Purpose**: A master list of every data category in your life. This ensures no information is lost during the Second Brain integration.

## 1. Human Core (The "Why")
- **Health**: Sleep logs, gym stats, diet.
- **Emotions**: Mood logs, emotional friend protocol journals.
- **Spirituality**: Meditations, connection logs.
- **Identity**: Personal constitution, birth chart, Vision 2034.

## 2. Technical & Academic (The "Mastery")
- **School**: Electrical Engineering lab reports, notes, assignments.
- **Builders**: Python scripts, CAD designs, automation flows (n8n).
- **Research**: Book summaries, web bookmarks, Perplexity reports.

## 3. Executive Layer (The "Doing")
- **Active Tasks**: Universal Inbox, Action Manager.
- **Project Tracking**: Checklists (v2.3), project handlers.
- **Execution**: VS Code workspace logic, Antigravity commands.

## 4. Financial & Administrative (The "Security")
- **Finance**: Budgeting, scholarships, income logs.
- **Admin**: Passport/ID copies, digital receipts, warranties.
- **Operations**: Passwords, version control backups.

## 5. Relational (The "People")
- **CRM**: Friends/Family birthdays, networking notes.
- **Communication**: Chat logs (Claude session archives).

## 6. Media & Creative (The "Assets")
- **Art/Design**: Dashboard mockups, aesthetic references.
- **Archive**: Photos, saved videos from social media.

---

## Data Management Rules
- **Obsidian First**: All finished data must eventually live in `My Brain`.
- **Dashboard Second**: Active data lives in the Eso Dashboard for 24-48 hours before archiving.
- **No Trash**: Only save data that can be used for future learning or automation.
